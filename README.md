# CultOfThreads

A portfolio website for [Cult of Threads](https://www.instagram.com/cultofthreads_)


---


### How to run - Development

- Install [bun](https://bun.sh/docs/installation#installing)
- Run `bun install` at the root of the repository
- Run `bun dev`


### Technologies Used

- Bun
- TypeScript
- Vite / React
- Framer Motion
- Jest
